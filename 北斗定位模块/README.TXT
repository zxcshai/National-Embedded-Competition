实验器材:
	NANO STM32F103开发板V1版本
	
实验目的:
	学习ATK-S1216F8-BD模块的使用，实现GPS/北斗定位
	
硬件资源:
	1,DS0(连接在PC0)
	2,串口1(波特率:115200)
	2,串口2(波特率:38400,PA2/PA3接GPS模块的RXD和TXD)
	3,按键KEY1(PC9)
	4,ATK-S1216F8-BD模块，同开发板的连接关系如下:
	  STM32开发板-->ATK-S1216F8-BD模块
			  PA2-->RXD
			  PA3-->TXD
			  GND-->GND
		  5V/3.3V-->VCC 
	
实验现象:
	本实验将实现如下功能：通过串口2连接ATK-S1216F8-BD模块，获取GPS信息，包括精度、纬度、高
	度、速度、用于定位的卫星数、可见卫星数、UTC时间等信息。可以通过USMART工具，设置GPS模块的刷新
	速率（最大支持20Hz刷新）和时钟脉冲的配置。通过KEY1按键，切换GPS解析好的信息打印显示，还是NMEA数据的上传（上位机显示）
	
注意事项:
     1,ATK-S1216F8-BD模块的波特率默认设置为38400。 
	 2,串口1 的通信波特率是115200。
	 3,GPS模块连接的天线必须放置于室外或者窗户旁边，否则有可能收不到GPS信号。
